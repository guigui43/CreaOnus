Instructions de Débogage

  1. Ouvrez les DevTools (F12)
  2. Allez dans l'onglet Console pour voir les erreurs
  3. Allez dans l'onglet Network pour voir les fichiers qui échouent
  4. Rechargez la page (Ctrl+F5) pour forcer le rechargement du cache

Erreurs communes:

  • 404 sur style.css → Fichier CSS manquant
  • MIME type incorrect → Problème de serveur
  • CORS error → Problème de sécurité
