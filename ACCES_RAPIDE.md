# 🚀 Guide d'Accès Rapide aux Thèmes CreaOnus

## 📱 URLs Simplifiées pour Serveur Local

Après avoir lancé votre serveur local avec `npx serve -l 3000` :

### 🎯 Sélecteur de Thèmes
```
http://localhost:3000/demo.html
http://localhost:3000/themes.html
```

### 🎨 Thèmes Directs
```
http://localhost:3000/elegant.html    # 💎 Thème Elegant
http://localhost:3000/neon.html       # 🌟 Thème Neon Gaming  
http://localhost:3000/minimal.html    # 🤍 Thème Minimal
http://localhost:3000/retro.html      # 📻 Thème Retro
http://localhost:3000/rainbow.html    # 🌈 Thème Rainbow
```

### 🏠 Site Original
```
http://localhost:3000/               # Style original
http://localhost:3000/index.html     # Style original
```

## 🔧 URLs Complètes (Alternative)

Si vous préférez les URLs complètes :

```
http://localhost:3000/variants/elegant/index.html
http://localhost:3000/variants/neon/index.html
http://localhost:3000/variants/minimal/index.html
http://localhost:3000/variants/retro/index.html
http://localhost:3000/variants/rainbow/index.html
```

## 🌐 Déploiement sur GitHub Pages

Remplacez `localhost:3000` par votre URL GitHub Pages :

```
https://[username].github.io/CreaOnus/rainbow.html
https://[username].github.io/CreaOnus/elegant.html
# etc...
```

## ❌ Erreurs Courantes

### URL Incorrecte
❌ `http://localhost:3000/rainbow` (404 - pas d'extension)
✅ `http://localhost:3000/rainbow.html`

### Chemins Non Trouvés
❌ `http://localhost:3000/variants/rainbow` (404 - dossier)
✅ `http://localhost:3000/variants/rainbow/index.html`

## 🎵 Test Rapide

1. **Démarrer le serveur** : `npx serve -l 3000`
2. **Ouvrir** : `http://localhost:3000/demo.html`
3. **Choisir un thème** et profiter du karaoké !
