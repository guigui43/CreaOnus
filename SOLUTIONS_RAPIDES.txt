🔧 SOLUTIONS RAPIDES - Thèmes sans style
==========================================

PROBLÈME: Les thèmes s'affichent en noir et blanc, sans couleurs ni style

🚀 SOLUTIONS PAR ORDRE DE PRIORITÉ:

1. FORCER LE RECHARGEMENT
   → Appuyez sur Ctrl + F5 (ou Cmd + Shift + R sur Mac)
   → Cela vide le cache et recharge tout

2. TESTER AVEC LE DIAGNOSTIC
   → Ouvrez: http://localhost:3000/test-css.html
   → Regardez les résultats en vert/rouge

3. CHANGER DE SERVEUR
   → Arrêtez le serveur actuel (Ctrl + C)
   → Essayez: python -m http.server 3000
   → Ou: npx http-server -p 3000

4. VÉRIFIER LES FICHIERS
   → Ouvrez: http://localhost:3000/check-files.html
   → Tous les fichiers doivent être en vert ✅

5. NAVIGATION PRIVÉE
   → Ouvrez Chrome/Firefox en mode privé/incognito
   → Testez: http://localhost:3000/rainbow.html

6. VÉRIFIER L'URL
   ❌ http://localhost:3000/rainbow (404)
   ✅ http://localhost:3000/rainbow.html (OK)

🔍 DIAGNOSTIC RAPIDE:
- Ouvrez F12 (DevTools) → Console
- Cherchez les erreurs en rouge
- Si vous voyez "404" pour style.css → Fichiers manquants
- Si vous voyez "MIME type" → Problème de serveur

📱 URLS DE TEST:
http://localhost:3000/demo.html         ← Page de choix des thèmes
http://localhost:3000/rainbow.html      ← Thème Rainbow direct  
http://localhost:3000/test-css.html     ← Test diagnostic
http://localhost:3000/check-files.html  ← Vérification fichiers

🆘 SI RIEN NE MARCHE:
1. Redémarrez votre ordinateur
2. Ouvrez VS Code → Extension "Live Server" 
3. Clic droit sur index.html → "Open with Live Server"
4. Ou double-cliquez directement sur les fichiers HTML

💡 ASTUCE: 90% des problèmes CSS sont résolus avec Ctrl + F5 !

Pour plus de détails: Consultez DEPANNAGE_CSS.md
